# 環境檢核表 (Environment Checklist)

請學員在完成環境搭建後，勾選以下項目並在 `assets/` 資料夾存入相關截圖。

## 1. 軟體安裝
- [ ] VS Code 已安裝
- [ ] PlatformIO IDE 插件啟動成功 (看到左側的外星人圖標 👽)

## 2. 首次編譯
- [ ] `src/main.cpp` 已修正 `#include` 邏輯
- [ ] 點擊下方狀態列的 Checkmark (✔) 編譯成功
- [ ] 終端機出現綠色的 `SUCCESS` 字樣

## 3. 功能驗證
- [ ] IntelliSense 正常運作 (滑鼠停在 `pinMode` 上可見函式定義)
- [ ] `platformio.ini` 已加入 `ArduinoJson` 依賴
- [ ] `platformio.ini` 已設定正確的 `monitor_speed`
